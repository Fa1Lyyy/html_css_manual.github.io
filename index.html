HTML-язык гипертекстовой разметки сайта

Элементы HTML являются блоками HTML страниц. С помощью HTML разные конструкции, изображения и другие объекты, такие как интерактивная веб-форма, могут быть встроены в отображаемую страницу. HTML предоставляет средства для создания заголовков, абзацев, списков, ссылок, цитат и других элементов. Элементы HTML выделяются тегами, записанными с использованием угловых скобок.

Возможности HTML

  • Делать текстовую разметку
  • Встраивать медиа
  • Создавать ссылки и навигацию
  • Создавать таблицы
  • Создавать формы

Аббревиатура CSS расшифровывается как Cascading Style Sheets, что в переводе означает «каскадные таблицы стилей». Это язык разметки, используемый для визуального оформления веб-сайтов.

Как и HTML, CSS на самом деле не является языком программирования. Это не язык разметки - это язык таблицы стилей. Это означает, что он позволяет применять стили выборочно к элементам в документах HTML.

Без CSS сайты выгдялели бы очень линейно, css позволяют оформить сайт в нужном стиле
